# 📰 Year 7 English – Mastering Passive Voice

An interactive Streamlit web app for Year 7 English students to learn and practise **Passive Voice** through a crime-scene news theme.

---

## 🖥️ What the App Does

The lesson is split into **6 interactive slides**:

| Slide | Topic |
|-------|-------|
| 1 | Introduction – The Scene of the Crime |
| 2 | Active vs. Passive: Grammar Basics |
| 3 | The Journalist's Secret – Why use it? |
| 4 | Quiz – 3 questions with hidden reveal |
| 5 | Headline Builder – drag-and-drop style |
| 6 | Writing Challenge – Be the Reporter! |

Answers and feedback are **hidden until the student clicks to reveal** — ideal for classroom use.

---

## 🚀 How to Run

### 1. Clone this repository
```bash
git clone https://github.com/YOUR-USERNAME/passive-voice-app.git
cd passive-voice-app
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the app
```bash
streamlit run passive_voice_app.py
```

The app will open automatically in your browser at `http://localhost:8501`

---

## 🛠️ Running in Mu Editor

1. Open **Mu Editor**
2. Click **Load** and select `passive_voice_app.py`
3. Open your computer's **Terminal / Command Prompt**
4. Navigate to the folder:
